# Playwright SauceDemo Framework

## Overview
This project is an automation testing framework built using Playwright and TypeScript for SauceDemo application testing.

## Tech Stack
- Playwright
- TypeScript
- Node.js
- Page Object Model (POM)

## Installation

```bash
npm install
```

## Install Browsers

```bash
npx playwright install
```

## Run Tests

```bash
npx playwright test
```

## Run Specific Test

```bash
npx playwright test tests/login.spec.ts
```

## Generate Report

```bash
npx playwright show-report
```
